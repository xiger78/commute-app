# 출퇴근 관리 앱

**프로그램 이름:** 출퇴근 관리 (Commute Manager / 出退勤管理)  
**버전:** 1.0.0  
**패키지 ID:** `com.commuteapp`

출근 날짜·유형 지정, 출퇴근 시각 입력, 출근 이력 확인, 일본 공휴일 조회, 설정(다국어·출근 유형·휴계시간·근태장표 CSV·메일 전송) 기능을 제공하는 React Native 모바일 앱입니다.

## 기능별 설명

앱은 **상단 귀여운 배너**와 배너 이미지 위 **6개 링크 버튼 메뉴**로 구성됩니다. 메뉴 이름은 **알림 · 날짜지정 · 출퇴시간 · 이력확인 · 올해휴일 · 설정** 입니다. 기본 표시 언어는 **일본어** 이며, 설정에서 **중국어·한국어·영어**로 변경할 수 있습니다.

**다른 언어 매뉴얼:** [日本語](README_JP.md) · [中文](README_ZH.md) · [English](README.md)

### 1. 알림 (출근일 메모)

출근일에 입력한 특이사항 메모를 날짜별 카드로 표시합니다.

**사용 방법:**
- **날짜지정**에서 출근일로 지정하고 **출퇴시간** 화면에서 메모를 입력·저장한 날만 표시
- 형식: `YYYY/MM/DD(요일):출근종류(출근시각)` / `메모:내용`
- 출근시각은 출퇴시간 저장값 우선, 없으면 설정의 출근 유형 기본 시각
- 예:
```
2026/06/11(목요일):정상출근(09:00)
메모:당일 혼방데이터 수정 릴리즈 작업있음
```
- 최신 날짜가 위에 표시

![알림](docs/images/en/screen-notifications.png)

### 2. 출근 날짜 지정

월간 달력에서 출근일과 출근 유형을 선택합니다.

**사용 방법:**
- **년·월** 을 출근 이력 확인 화면과 동일한 드롭다운 선택기로 지정
- 하단 **출근 유형 버튼** (통상·일찍·늦게·재택·휴가) 중 하나를 선택한 뒤 날짜를 탭하여 지정
- 지정된 날짜는 선택한 유형의 **설정 색상** 으로 표시
- 같은 날짜를 빠르게 두 번 탭하면 해제
- **일본 공휴일** 은 달력에 **빨간 동그라미** 로 표시
- 제목 오른쪽 **초기화** 로 해당 월 출근일 전체 해제
- 범례: 5가지 출근 유형 색상 + 빨간색 = 휴일

**출근 유형 기본값 (설정에서 변경 가능):**
| 유형 | 기본 출근 | 기본 퇴근 |
|------|-----------|-----------|
| 통상 | 08:40 | 출근 + 8시간 |
| 일찍 | 06:00 | 16:00 |
| 늦게 | 11:00 | 20:00 |
| 재택 | 08:40 | 출근 + 8시간 |
| 휴가 | — | — |

![출근 날짜 지정](docs/images/en/screen-work-date.png)

### 3. 출퇴근 시각 입력

선택한 월의 각 날짜별 출퇴근 시각을 입력합니다.

**사용 방법:**
- **년·월** 을 출근 이력 확인 화면과 동일한 드롭다운 선택기로 지정
- **일괄적용** 영역에서 출근·퇴근 시각 입력 후 **적용하기** 와 **저장** 버튼을 나란히 탭
- 날짜별로 컴팩트 **HH시 MM분** 입력 UI로 개별 수정
- 각 날짜 **메모** 항목에 특이사항 입력 가능
- 타이틀 오른쪽 **다시설정** 버튼으로 해당 월 전체 시간·메모를 **00:00** / 빈 값으로 초기화
- **저장** 후 저장 영역 아래 미리보기 목록 표시

**색상 범례 (일괄적용 아래):**
- **일찍출근 · 통상출근 · 늦게출근 · 재택근무** — 설정의 출근 유형 색상과 동일

**저장 미리보기 (출근 이력 확인과 동일 형식):**
- **첫 줄:** `[가동시간:합산시간]` — 저장된 일별 가동시간 합계 (소수 1자리)
- 각 행: `YYYY/MM/DD(요일) HH:MM-HH:MM (가동시간)` **가운데 정렬**
- 메모가 있으면 `[메모:내용]` 추가
- 가동시간 = 퇴근 − 출근 − **아침·점심·저녁 휴계시간** (설정 참고), 괄호 안 소수 (`9.0`, `9.5`)
- **휴가** 일은 가동시간 합산에서 제외

**날짜 표시 및 카드 색상:**
- 각 행은 `YYYY/MM/DD(요일):유형` 형식
- 카드 색상은 **설정의 출근 유형 색상** 으로 표시 (통상·일찍·늦게·재택·휴가)
- **평일, 달력 미지정** → `:재택`
- **토·일·공휴일, 미지정** → `:공휴일` (회색 카드)
- **토·일·공휴일, 출근일 지정** → 기본 `:출근`; 날짜 라벨(▼) 탭으로 **출근/재택** 전환
- **토·일·공휴일 출근** 시 기본 시각 **08:40–17:40** (개별 수정 가능)

**일괄적용 규칙:**
- **통상·재택** 평일에만 적용 (**일찍·늦게·휴가는 수동 수정만**)
- **토·일·일본 공휴일(祝日) 제외** — 고정 공휴일, 해피 먼데이, 춘분·추분, 대체 공휴일, 국민의 휴일
- 화면에 `토·일 및 일본 공휴일 제외 · 적용 대상 N일` 안내

![출퇴근 시각 입력](docs/images/en/screen-commute-time.png)

### 4. 출근 이력 확인

월간 출근 이력을 확인합니다.

**사용 방법:**
- 드롭다운으로 년·월 선택
- 선택한 월의 목록이 자동으로 표시됨
- **첫 줄:** `[가동시간:합산시간]` — 해당 월 일별 가동시간 합계
- 각 행: `YYYY/MM/DD(요일) HH:MM-HH:MM (가동시간)` 형식, **가운데 정렬**
- 괄호 안 가동시간은 저장 미리보기와 동일 (점심·저녁 휴계시간 제외, `9.0` / `9.5` 형식)
- 예:
```
[가동시간:160.0]
2026/06/03(수) 09:00-18:00 (8.0)
2026/06/04(목) 09:00-18:00 (8.0)
```
- 카드 색상·유형 표시는 출퇴근 시각 화면과 동일 (출근 유형별 색상)

![출근 이력 확인](docs/images/en/screen-attendance-history.png)

### 5. 올해 휴일

일본 공휴일을 년·월 단위로 확인합니다.

**사용 방법:**
- **초기화** 로 년·월 선택 해제
- ◀ ▶ 로 년도 선택 (처음은 미선택)
- 년도만 선택 시 1년치 공휴일 목록: `YYYY/MM/DD(요일):공휴일명`
- 월까지 선택 시 해당 월 달력 + 공휴일 표시
- 대체휴일·국민의 휴일 구분 표시

![올해 휴일](docs/images/en/screen-year-holidays.png)

### 6. 설정

화면표시언어, 출근 유형, 휴계시간, 근태장표출력(CSV), 메일 보내기를 설정합니다.

#### 6-1. 화면표시언어
**일본어 · 중국어 · 한국어 · 영어** 순으로 선택. 모든 화면이 즉시 변경됩니다.

#### 6-2. 출근 유형·색상·시각
- **통상 / 일찍 / 늦게 / 재택** — 표시 색상과 출근 시각 설정 (저장 시 퇴근 시각 자동 계산, 일찍·늦게는 고정 퇴근 시각 유지)
- **휴가** — 표시 색상만 설정 (가동시간 계산 제외)
- 카드 하단 **저장** 버튼(전체 폭)으로 일괄 저장

#### 6-3. 휴계시간 설정
- 카드: **휴계시간설정(아침,점심,저녁)** (카테고리: 휴계시간 설정)
- **아침휴식 (가동시간 제외)** — 기본값 **1시간**; 출근 시각이 **08:00 미만** 일 때만 차감
- **점심휴식 (가동시간 제외)** — 기본값 **1시간** (`01:00`); **휴가** 일은 제외
- **저녁휴식 (가동시간 제외)** — 기본값 **0시간** (`00:00`)
- 각 항목 **시·분** 독립 입력 (HH시 MM분), 하단 **저장** 버튼으로 한 번에 저장
- 저장된 휴계시간은 이력확인·저장 미리보기·CSV 출력 시 가동시간에서 합산 제외

#### 6-4. 근태장표출력(CSV)
- 출력할 달 선택
- **출력** 버튼(전체 폭)으로 CSV 파일 생성 및 공유 (§6-3 휴계시간·§6-2 출근 유형 반영)

**CSV 출력 형식 예시:**
```
2026년 06월 출근 이력
01일: 출근시각:09:00、퇴근시각:18:00、가동시간:08시간00분
...
[총근무시간:160시간00분]
```

#### 6-5. 메일 보내기
- 받는 주소, 제목, 본문 입력
- 파일 첨부 (출력한 CSV도 자동 첨부 가능)
- **메일 보내기** 버튼(전체 폭)으로 기기 메일 앱 실행
